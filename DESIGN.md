# Spanish Lab · DELE B2 디자인 원칙

이 문서는 현재 UI의 시각·상호작용 계약을 기록한다. 따뜻한 학습 노트 같은 편집 디자인을 유지하면서, 긴 학습 흐름은 탐색과 집중을 우선한다.

## 시각 언어

- 배경은 종이색 `--bg`, 콘텐츠 면은 `--surface`, 본문은 남색 계열 `--ink`, 핵심 행동과 상태는 `--accent`를 사용한다.
- accent 위 텍스트는 반드시 `--on-accent`를 사용한다. light/dark 모두 일반 텍스트 대비 4.5:1 이상을 유지한다.
- 제목은 Georgia 계열 serif, 본문과 조작 UI는 시스템 sans-serif를 사용한다. 새 웹폰트 의존성은 추가하지 않는다.
- 장식은 큰 B2 워터마크, 얇은 선, 제한된 accent로 끝낸다. 기능과 무관한 일러스트·아이콘·그라데이션은 추가하지 않는다.

## 레이아웃과 간격

- 공통 콘텐츠 폭은 `--max: 1180px`, 본문 읽기 폭은 최대 70–75ch다.
- 페이지 구획은 `page-section`, 선택 가능한 독립 객체는 `card`, 단순 통계·설명 구획은 선과 여백을 사용한다.
- sticky 요소는 `--header-height`, `--context-bar-height`, `--scroll-offset` 계약을 공유한다. 각 화면에서 별도 하드코딩한 top 값을 만들지 않는다.
- 모바일에서는 한 번에 한 영역과 하나의 주 행동을 우선 노출한다. 긴 카드 그리드를 그대로 축소하지 않는다.

## 컴포넌트 상태

- 버튼은 primary, secondary, small 세 단계만 사용한다. coarse pointer와 모바일에서 조작 영역은 최소 44×44px이다.
- loading은 실제 0건과 구분되는 skeleton/문구를, empty는 다음 행동을, error와 비영속 저장은 복구 방법을 함께 보여준다.
- 진행 상태는 `not-started`, `in-progress`, `done` 용어를 공유하고 색상만으로 구분하지 않는다.
- 카드 전체가 이동 대상일 때만 block link를 사용한다. 카드 내부에 여러 행동이 있으면 명시적 버튼을 둔다.

## 접근성과 반응형

- 모든 기능은 키보드와 터치만으로 완료할 수 있어야 하며 drag는 보조 입력으로만 허용한다.
- 현재 위치는 `aria-current`, 토글은 `aria-pressed`, 비동기 상태는 적절한 live region으로 전달한다.
- dialog는 열기 버튼으로 포커스를 복원하고 Escape로 닫힌다. reduced-motion 환경에서는 부드러운 스크롤과 전환을 강제하지 않는다.
- 320px, 390px, 768px, 1440px에서 가로 overflow와 sticky 가림을 확인한다.

## 콘텐츠와 신뢰

- 공식 자료와 창작 연습문항을 문구와 배지로 명확히 구분한다.
- 진도·시험 기록은 기기 로컬 저장임을 숨기지 않는다. 손상 복구·비영속 fallback·부분 가져오기는 성공 상태와 구분한다.
- 한국어 안내를 기본으로 하되 시험명·Tarea·지문은 스페인어 원문 의미를 보존한다.
